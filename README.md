# 🌍 Trip Planner

A modern and responsive travel planning web application that helps users discover destinations, plan trips, estimate budgets, and organize travel information in one place.

## 🔗 Live Demo

👉 https://muthamizh499.github.io/trip-planner/

---

## 📖 Overview

Trip Planner is a web-based application designed to simplify travel planning. Users can browse destinations, explore attractions, estimate travel expenses, and organize their trips through an intuitive interface.

The application focuses on providing a clean user experience with responsive design, making it accessible across desktop and mobile devices.

---

## ✨ Features

- 🗺️ Explore popular travel destinations
- 📍 View destination details
- 💰 Travel budget estimation
- 🏨 Hotel and accommodation suggestions
- 📅 Trip planning interface
- 💬 Interactive travel chatbot
- 📱 Fully responsive design
- ⚡ Fast and lightweight

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/muthamizh499/trip-planner.git
```

### Navigate to the Project

```bash
cd trip-planner
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
npm start
```

The application will be available at:

```
http://localhost:3000
```
